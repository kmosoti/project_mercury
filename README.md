# Mercury - The Ultimate Discord Bot

Introducing Mercury, the upcoming Discord bot that promises to take your server management experience to the next level. Although still a work in progress, Mercury has a lot in store for its users.

In addition to the features already mentioned, such as customizable commands, server statistics, and moderation tools, Mercury also plans to implement:
- AI chat feature
- DJ mode for music playback
- A suite of games (TBD)

Mercury also plans to introduce a tier pricing system, with a free tier supported by ads, and a premium tier that offers more advanced features for a monthly subscription.

Designed to be user-friendly, Mercury will have a sleek and modern interface that makes it easy to navigate and use for both experienced server administrators and casual users.

Keep an eye out for updates as Mercury continues to evolve and grow, offering more features and benefits to help you make the most out of your Discord server.
